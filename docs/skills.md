## Available Skills

This project has custom skills in `.claude/skills/`. Use these skills for specific tasks:

- `/server-architecture`
  - **When to use**: Implementing or designing server APIs, adding new endpoints
  - **Key information**: Four-layer architecture (Presentation, Orchestration, Logic, Data), testing strategy per layer
- `/database-design`
  - **When to use**: Designing database schemas, creating tables, modeling data
  - **Key information**: Event/Resource classification, proper state management
- `/test`
  - **When to use**: Writing tests, designing testable functions, discussing test strategy
  - **Key information**: BDD, Gherkin, Google test size classification (Small/Medium/Large)
- `/create-pr`
  - **When to use**: Creating pull requests
  - **Key information**: Project conventions, issue linking
