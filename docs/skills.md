## Available Skills

This project has custom skills in `.claude/skills/`. Use these skills for specific tasks:

- `/server-architecture` - Server API implementation following four-layer testable architecture (Presentation, Orchestration, Logic, Data)
- `/database-design` - Database schema design using Event/Resource classification and proper state management
- `/test` - Writing testable code and tests using BDD, Gherkin, and Google test size classification
- `/create-pr` - Create pull requests following project conventions
